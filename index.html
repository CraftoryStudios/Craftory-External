Features

Custom Blocks

Adding blocks to the game including ores, machines and more. And all these blocks are real, not tile entitys or any other tricks, meaning they don't cause any extra lag.

Adds loads of cutom items. bringing new elements to the game, more ingots and tools that can be used to interact with machines.
