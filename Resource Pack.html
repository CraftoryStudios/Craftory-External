Resource Pack

FAQ

Overview

One of the core features of Craftory is it's creative usage of resource packs to aid in creating an extraordinary experience for it's users. To help you get the most of this this page includes a resource pack FAQ and resource pack change log!

1. How can I use Craftory along side another server side resource pack?

If your server already has a resource pack, you can download Craftory resource pack bellow and merge it with your current one. This is a manual process and if done wrong can cause issues. There are many tutorials online to help and also our Discord server.

Once the packs are merged you can go into plugins/Craftory/config.yml and set forcePack to false.
This will mean Craftory will no longer provide the resource pack to your users. You can then host it yourself, we recommend https://mc-packs.net/ (as its free).

Last set is to put the hosted pack link in server properties. There are many tutorials on setting up server resource packs!

2. How can I use Craftory along side another client side resource pack?

Client side resource packs can be used along side Craftory and in most cases they shouldn't cause any issues. If they do please report this to us and we will do our best to fix it!

3. How do I update the resource pack?

If you are just using standard Craftory pack hosting and client side add-on packs, you should be sorted. The Craftory resource pack is automatically updated each version.

Merged Server Packs:
If you have merged with your own resource pack you can either re-merge it again with the updated Craftory pack, or use the change log links ago to just change, add or remove the correct files.
